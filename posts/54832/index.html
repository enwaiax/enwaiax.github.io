前言

RainLoop 是一个简单，现代，快速和开源的基于 Web 的电子邮件客户端，用 PHP 编写。

Rainloop 特性

  • 现代的 Web 界面，允许上传自定义主题
  • 快速轻便
  • 同时支持 SMTP 和 IMAP
  • 支持多个帐户，可以从一个地方访问所有电子邮件地址。
  • 支持双因素身份验证。
  • 与 Facebook, Twitter, Google, and Dropbox 集成
  • 无需数据库， 可以直接访问邮件服务器
  • 易于安装和升级（在管理面板中一键升级）

Rainloop 版本选择

Rainloop 有标准版和社区版两个版本， 本文使用社区版(免费)
